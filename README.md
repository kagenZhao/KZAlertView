# KZAlertView
 KZAlertView, UI基于 FCAlertView, 使用Swift 重构 FCAlertView

## Installation

KZAlertView is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'KZAlertView'
```
